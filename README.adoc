*Unified MCP adapters for 28 static site generators across 12+ programming languages.*

Sparkle-SSG is a satellite implementation in the hyperpolymath ecosystem, providing security-hardened adapters for integrating static site generators with AI assistants via the Model Context Protocol (MCP).

== Quick Start

[source,bash]
----
# Clone repository
git clone https://github.com/hyperpolymath/sparkle-ssg
cd sparkle-ssg

# Initialize development environment
just init

# Run checks
just check

# Run tests
just test
----

== Features

* **28 SSG Adapters** - Rust, Haskell, Elixir, Julia, Clojure, and more
* **Security Hardened** - Input validation, path traversal prevention, command injection protection
* **MCP Compatible** - Integrates with AI assistants via Model Context Protocol
* **RSR Gold Compliant** - SPDX headers, SHA-pinned actions, signed commits
* **Full Test Suite** - Unit, integration, and security tests
* **Container Ready** - Containerfile for Docker/Podman deployment

== Usage

=== As a Module

[source,javascript]
----
import { getAdapter, listAdapters, metadata } from '@hyperpolymath/sparkle-ssg';

// List available adapters
console.log(listAdapters());
// ['zola', 'hakyll', 'cobalt', ...]

// Load and use an adapter
const zola = await getAdapter('zola');
console.log(zola.name);     // "Zola"
console.log(zola.language); // "Rust"

// Execute a tool
const result = await zola.tools[0].execute({ path: './my-site' });
----

=== Via Just Commands

[source,bash]
----
just adapters          # List all adapters
just validate-adapter zola  # Validate specific adapter
just test              # Run tests
just build             # Build all
----

== Security

Sparkle-SSG implements comprehensive security measures:

* **Input Validation** - All inputs validated before execution
* **Path Traversal Prevention** - Blocks `../` escape attempts
* **Command Injection Prevention** - Rejects shell metacharacters
* **CodeQL Scanning** - Automated security analysis
* **Dependabot** - Dependency vulnerability monitoring

See link:SECURITY.md[SECURITY.md] for vulnerability reporting.
